# HealthCompanion

**HealthCompanion** is an educational, AI-powered health triage companion designed to help users explore symptoms, understand general medical concepts, and learn about potential healthcare pathways. By leveraging the fast Groq API (running Llama 3) for natural language reasoning and the browser's native Web Speech API for voice interactions, users can seamlessly ask questions and receive spoken responses in a highly interactive format.

## Technical Stack

- **Backend**: Python, FastAPI, Groq API (Llama 3), python-dotenv, Uvicorn
- **Frontend**: React (Create React App), Tailwind CSS, browser Web Speech API for voice synthesis and recognition
- **Storage**: Browser LocalStorage for persistent local chat logs

## Pushing to GitHub

To push the entire monorepo project safely to GitHub, follow the steps below from the **workspace root** directory:

### Step 1: Clean Nested Git Folders
Since Create React App automatically initializes a nested `.git` folder inside the `health-frontend/` directory, you must delete it. If you don't, GitHub will treat the frontend as a submodule and it will show up as an empty gray folder on GitHub.

* **PowerShell**:
  ```powershell
  Remove-Item -Recurse -Force health-frontend\.git
  ```
* **Command Prompt (CMD)**:
  ```cmd
  rmdir /s /q health-frontend\.git
  ```
* **macOS/Linux**:
  ```bash
  rm -rf health-frontend/.git
  ```

### Step 2: Initialize Git and Commit
```bash
# Initialize a Git repository in the root workspace folder
git init

# Stage all files (will respect the root .gitignore)
git add .

# Create the initial commit
git commit -m "Initial commit: HealthCompanion web app"
```

### Step 3: Publish to GitHub
1. Create a new empty repository on your GitHub account.
2. Link your local repository to GitHub and push:
```bash
# Rename default branch to main
git branch -M main

# Add your GitHub remote repository (replace with your actual URL)
git remote add origin https://github.com/your-username/health-companion.git

# Push to GitHub
git push -u origin main
```
